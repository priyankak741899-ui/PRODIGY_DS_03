# PRODIGY_DS_03

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service using the Bank Marketing Dataset.

Tools Used:

* Python
* Pandas
* Scikit-learn
* Google Colab

Model Accuracy: 87.05%

Key Steps:

* Data loading and preprocessing
* Encoding categorical variables
* Train-test split
* Decision Tree model training
* Model evaluation using accuracy score
